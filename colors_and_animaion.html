colors and animation

solution description

If you see a red sign, I think you would stop short and just see if there was danger in danger. Colors help to draw one's eyes, for example to signal danger or an important button. But colors can also help to create depth, where warm colors feel closer and cold colors feel further away, but create a hekariki, as you see the warm color first and the cold afterwards. In terms of color, animation is a good thing, which helps to create a flow that gives some life to a website. If a user holds the mouse over a button, then it makes sense that buttons change a bit so that the user has no doubt that he is on the button. Animation can also provide something exciting to watch, giving the user a bold experience of using the website. Colors and animation are an important thing that I have used in my solution to create a more exciting and intrusive website.
