design with frameworks

solution description

It takes a lot of time to build a website, because you have to make it responsive, style it and much more. The internet has been around for many years and that is why there are already over 100 of websites. But things that are used a lot are developed and made better all the time. This is where frameworks come into the picture, as they are a way that make it easier to implement things, faster, as they often come with shortcuts, different elements and features. This means you do not have to start from scratch every time you start creating a new website. In my solution I have used the framework Pico, to make my website responsive. The smart thing about pico is that it automatically adds a dark theme and styling to the elements without doing anything in css. Of course, using a framework always has its pros and cons.
